# ✊✋✌️ Rock Paper Scissors Game

A simple Python command-line game where the user plays **Rock, Paper, Scissors** against the computer.

The computer randomly selects an option, and the winner is decided based on standard game rules.

---

## 🚀 Features

* Interactive gameplay
* Random computer choices
* Score tracking system
* Unlimited rounds
* Exit option using `Q`

---

## 🛠 Technologies Used

* Python
* Random Module
